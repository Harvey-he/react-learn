React学习总结
----

###示例查看
1.安装依赖
```bash
npm install
```
2.运行gulp，开启一个静态服务器
```bash
gulp
```
3.访问以下地址，查看示例  
[http://localhost:8080/examples/](http://localhost:8080/examples/)
